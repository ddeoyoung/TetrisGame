# TetrisGame

Win32 API + GDI 로 만든 테트리스 게임

## 조작법

| 키 | 동작 |
|----|------|
| `←` `→` | 좌우 이동 |
| `↑` | 회전 (시계방향) |
| `↓` | 소프트 드롭 |
| `Space` | 하드 드롭 |
| `H` | 현재 블록 홀드 |
| `R` | 게임오버 후 재시작 |

## 게임 규칙
### 블록
- I, O, T, S, Z, J, L 7종의 테트로미노가 나온다

### 점수

| 조건 | 점수 |
|------|------|
| 1줄 클리어 | 100 × 레벨 |
| 2줄 클리어 | 300 × 레벨 |
| 3줄 클리어 | 500 × 레벨 |
| 4줄 클리어 | 800 × 레벨 |
| 소프트 드롭 | 1칸당 1점 |
| 하드 드롭 | 1칸당 2점 |

### 레벨

- 10줄을 지울 때마다 레벨이 올라간다. 
- 레벨이 오르면 블록 낙하 속도가 빨라진다.
