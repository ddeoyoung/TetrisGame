# TetrisGame

Win32 API + GDI 로 만든 테트리스 게임\
싱글 플레이, 멀티 플레이, 리플레이를 지원합니다.

## 화면 구성

타이틀 화면에서 세 가지 모드를 선택합니다.

| 메뉴 | 설명 |
|------|------|
| `SINGLE PLAY` | 싱글 플레이 |
| `MULTI PLAY` | 1:1 멀티 플레이 |
| `REPLAY` | 저장한 리플레이 파일 재생 |

- `↑` `↓` : 메뉴 이동  `Enter` : 선택  `ESC` : 뒤로/타이틀 복귀

## 조작법

| 키 | 동작 |
|----|------|
| `←` `→` | 좌우 이동 |
| `↑` | 회전 (시계방향) |
| `↓` | 소프트 드롭 |
| `Space` | 하드 드롭 |
| `H` | 현재 블록 홀드 |
| `R` | 게임오버 후 재시작 |
| `ESC` | 타이틀로 나가기 |

---
## 게임 규칙
### 블록
- I, O, T, S, Z, J, L 7종의 테트로미노가 나옵니다.
- *홀드*는 한 블록당 한 번만 사용할 수 있습니다.

### 점수

| 조건 | 점수 |
|------|------|
| 1줄 클리어 | 100 × 레벨 |
| 2줄 클리어 | 300 × 레벨 |
| 3줄 클리어 | 500 × 레벨 |
| 4줄 클리어 | 800 × 레벨 |
| 소프트 드롭 | 1칸당 1점 |
| 하드 드롭 | 1칸당 2점 |

### 레벨

- 10줄을 지울 때마다 레벨이 올라간다. 
- 레벨이 오르면 블록 낙하 속도가 빨라진다.

## 멀티 플레이 (1:1)
같은 네트워크에 있는 두 PC 가 TCP로 연결해 대전 가능합니다. (기본 포트 7777)

### 접속 방법
1. 호스트는 `MULTI PLAY → HOST` 로 방 생성
2. 게스트는 `MULTI PLAY → JOIN` 선택 후 호스트의 IP 입력 (숫자와 '.' 만 입력)
3. 연결되면 게임 시작

## 리플레이
### 저장
싱글/멀티 게임이 끝나면 `Save Replay?` 창이 뜬다.
`예` 를 누르면 `.rep` 파일로 저장된다.

### 재생
타이틀 화면에서 `REPLAY` 메뉴를 골라 `.rep` 파일을 연다.
멀티 리플레이는 두 플레이어(P1/P2) 의 보드를 나란히 보여준다.

| 키 | 동작 |
|----|------|
| `Space` | 재생 / 일시정지 |
| `1` `2` `3` `4` | 배속 (0.5× / 1× / 2× / 4×) |
| `ESC` | 재생 종료 |
